# py-random-useragent

Generate a random User Agent

### Installation

```
pip install py-random-useragent
```

### Usage

``` python
>>> from py_random_useragent import UserAgent

>>> UA = UserAgent()
>>> print(UA.get_ua())
Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:109.0) Gecko/20100101 Firefox/114.0
```
